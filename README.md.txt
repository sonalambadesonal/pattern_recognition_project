How to Run Code-
Code Language- Python
Modules required - Tensorflow, keras, numpy, scikit-learn, matplotlib, opencv and imutils.
You can install all the modules using pip. 
After that-
1) Download MNISt Dataset From link https://www.kaggle.com/datasets/scolianni/mnistasjpg.
2) In the 3rd cell of jupyter-notebook change img_path ="path of training sample of dataset".
3) Now run all the cells one by one.
